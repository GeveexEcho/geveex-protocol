# Geveex Protocol Overview
Geveex is a long-term liquidity infrastructure on the Solana network. 

## Vision
To create a predictable, transparent, and community-aligned ecosystem using GVX tokens and strategic NFT instruments.

## Core Pillars
1. **Transparency:** All on-chain actions are verifiable.
2. **Fixed Supply:** Absolute cap of 21,000,000 GVX.
3. **Liquidity Focus:** Every phase is designed to compound liquidity.
